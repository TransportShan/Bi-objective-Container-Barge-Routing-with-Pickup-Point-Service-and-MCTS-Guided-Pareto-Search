# Data and Code Availability

A GitHub-ready reproduction package has been prepared for this study. It contains the instance-construction scripts, algorithm source code, processed benchmark test sets, numerical result tables, final figures, checksums, and reproduction commands.

The public repository URL and permanent archival DOI should be inserted into the manuscript after the repository is created and archived. Until then, use the following placeholder in the manuscript:

> A GitHub-ready reproduction package has been prepared for this study. It contains the instance-construction scripts, algorithm source code, processed benchmark test sets, numerical result tables, final figures, checksums, and a README with reproduction commands. The package will be made publicly available in a GitHub repository and archived with a permanent DOI before publication; the repository URL and DOI will be inserted in the final article after deposit.
